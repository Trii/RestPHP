Adds a response header to be sent

Arguments 2-N are imploded with '; ' to allow easier setting of headers like:

$response->addHeader('Content-Disposition', 'attachment', 'filename=fname.ext');

Which is output as: Content-Disposition: attachment; filename=fname.ext
